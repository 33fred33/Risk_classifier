Machine Learning based Classifier

Usage Guide:

In order to use Machine Learning based Classifier on your data, you need to follow these steps:

  1. Firstly in dataset tab, specify the dataset to be used. There is a default dataset "GermanCredit.Rda" for the first time users. The user can also add their own dataset in the format as specified in 'File input'.
  2. After selecting the dataset, the user can also view the dataset added in the 'View Dataset' tab.
  3. Once the dataset is added, the user can use the algorithm by clicking the 'calculate' button in 'Classifier' tab and can view the decision tree and various associative results with it in the tab.

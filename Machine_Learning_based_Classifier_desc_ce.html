Algorithm background

  1. Step1: - Feature Selection (can be done for prediction accuracy, model interpretability and multi-collinearity) :

    • Shrinkage : Fitting a model with all the 'p' predictors and estimated coefficients are shrunken towards zero so known as regularization has the effect of reducing variance. E.g. Lassso or Ridge Regression.
      • When to use them:
      • Prediction Accuracy: especially when p>n, to control the variance.
      • Model Interpretability: To remove irrelevant features, i.e. by setting the corresponding coefficients estimates to zero, we can obtain a model that is more easily interpreted.
    • Multi-collinearity between predictors variables : Identify the idependent variables in a model which are correlated, but to have a more interpretable model, you should avoid features that have a very high (~RÂ² > .8) being contained in the features.
      • Note : In this app, if the user is not interested in feature selection step they can directly go to Step 2 i.e. Possible subset of decision trees.
